・プログラムについて
このプログラムはPython3で書かれています．
ライブラリとしてpandasを用いています．
ログファイルを１行ごとに読み込み，要素ごとに分割してpandasのDataFrameにまとめています．
このDateFrameを元にpandasの関数を用いて集計を行なっています．

・実行手順
１．aikawa_log.pyを対象のログファイルと同じディレクトリに配置します．
２．コンソールで対処ディレクトリへ移動し，"python3 aikawa_log" で実行します．
３．対象とするログファイルの数の入力を求められるので半角で数字のみ入力します．
４．対象とするログファイルの名前の入力を求められるので，正しく入力します（対象のファイル数の分繰り返します）．
５．ファイルの読み込みが始まります．データサイズによってはかなりの時間がかかってしまいます．
６．集計を行う対象の期間を秒単位で指定します．期間の始点と終点が，年，月，日，時，分，秒と入力が求められるので，半角数字で入力します．
７．対象期間中の総アクセス回数が表示されます．
８．リモートホストごとにアクセス件数を降順に表示します．上から何件目まで表示するか，入力が求められるので，半角数字で入力します．
９．リモートホストごとのアクセス件数が８．で入力した任意の件数まで表示されます．
１０．プログラムが終了します．
